# Bot Example

Example source code for O<sub>byte</sub> chatbot.  Copy this bot and start adding your stuff.

## Requirements

node.js 6+

## Install
```
npm install
```
## Make sure you have done 
```
cp .env.testnet .env
```

## Run
```
node start.js
```
